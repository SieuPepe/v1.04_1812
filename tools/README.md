# Herramientas y Scripts Auxiliares

Este directorio contiene scripts auxiliares y herramientas de desarrollo para HydroFlow Manager.

## Scripts de Generación de Informes

### `generar_ejemplos_informes.py`
Genera informes de ejemplo para documentación y testing.

**Uso:**
```bash
python3 tools/generar_ejemplos_informes.py
```

### `generar_informes_completos.py`
Genera informes completos con todos los datos disponibles.

**Uso:**
```bash
python3 tools/generar_informes_completos.py
```

### `generar_todos_informes_exhaustivo.py`
Generación exhaustiva de todos los tipos de informes disponibles.

**Uso:**
```bash
python3 tools/generar_todos_informes_exhaustivo.py
```

## Utilidades

### `ejecutar_limpieza.py`
Script para ejecutar limpieza de datos y archivos temporales.

**Uso:**
```bash
python3 tools/ejecutar_limpieza.py
```

---

**Nota:** Estos scripts son para desarrollo y testing. No incluirlos en la distribución de producción.
