# Documentación de Desarrollo

Este directorio contiene documentación técnica para desarrolladores de HydroFlow Manager.

## Análisis y Especificaciones

### `ANALISIS_EXHAUSTIVO_INFORMES.md`
Análisis detallado del sistema de informes, incluyendo:
- Estructura de datos
- Tipos de informes disponibles
- Filtros y agrupaciones
- Casos de uso

### `ANALISIS_EXHAUSTIVO_COMPLETO.md`
Análisis completo de toda la aplicación, cubriendo:
- Arquitectura general
- Módulos principales
- Flujos de datos
- Dependencias

## Guías de Implementación

### `INSTRUCCIONES_IMPORTACION.md`
Instrucciones detalladas para importar datos desde Access:
- Estructura de datos de origen
- Proceso de migración
- Validación de datos importados
- Solución de problemas comunes

### `README_BUILD.md`
Guía para compilar y empaquetar la aplicación:
- Requisitos previos
- Configuración de PyInstaller
- Proceso de compilación
- Testing del ejecutable

## Resolución de Problemas

### `PROBLEMA_Y_SOLUCION.md`
Registro de problemas encontrados durante el desarrollo y sus soluciones:
- Problemas de base de datos
- Issues de interfaz
- Bugs conocidos y parches aplicados
