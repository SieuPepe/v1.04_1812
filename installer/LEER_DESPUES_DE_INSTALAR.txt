PRÓXIMOS PASOS
================================================================================

1. CONFIGURAR LA CONEXIÓN A MYSQL (Requerido)

   El asistente de configuración se ejecutará automáticamente al cerrar
   este instalador (si no desmarcó la opción).

   En el asistente deberá proporcionar:
   - Host de MySQL (generalmente 'localhost')
   - Puerto (por defecto: 3307)
   - Usuario y contraseña de MySQL
   - Nombres de los esquemas

   Si no ejecuta el asistente ahora, puede ejecutarlo más tarde desde:
   Menú Inicio → HydroFlow Manager → Configurar HydroFlow Manager


2. EJECUTAR HYDROFLOW MANAGER

   Una vez configurada la conexión a MySQL, puede ejecutar la aplicación:

   - Desde el escritorio: doble clic en el icono "HydroFlow Manager"
   - Desde menú inicio: Inicio → HydroFlow Manager
   - Desde la carpeta de instalación: HydroFlowManager.exe

TROUBLESHOOTING - PROBLEMAS COMUNES
================================================================================

Error: "No se puede conectar a MySQL"
  → Verifique que MySQL esté corriendo (Servicios de Windows)
  → Verifique usuario y contraseña
  → Verifique el puerto (3307 por defecto, puede ser 3306)

Error: "Esquemas no encontrados"
  → Los esquemas deben existir ANTES de ejecutar la aplicación
  → Verifique que manager, proyecto_tipo y cert_dev existan en MySQL

La aplicación no se inicia
  → Verifique que el archivo .env existe y está configurado
  → Ejecute el asistente de configuración nuevamente
  → Consulte los logs de error

El antivirus bloquea la aplicación
  → Agregue HydroFlow Manager a la lista de excepciones
  → Esto es un falso positivo común en aplicaciones PyInstaller


RECONFIGURAR O REINSTALAR
================================================================================

Para reconfigurar la conexión a MySQL:
  Ejecute: {carpeta_instalacion}\HydroFlowManager_Config.exe

Para reinstalar completamente:
  1. Desinstale desde: Panel de Control → Programas → HydroFlow Manager
  2. Vuelva a ejecutar este instalador


DESINSTALACIÓN
================================================================================

Para desinstalar HydroFlow Manager:
  Panel de Control → Programas y características → HydroFlow Manager → Desinstalar

O desde el menú inicio:
  Inicio → HydroFlow Manager → Desinstalar HydroFlow Manager

NOTA: El archivo .env (con credenciales) se eliminará automáticamente.
